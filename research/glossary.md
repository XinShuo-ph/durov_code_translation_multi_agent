## Style Notes

### Russian Names
- Use first name + patronymic in formal contexts
- Use first name alone in casual contexts
- Keep diminutives when they appear (Слава for Вячеслав)

### Titles
- Keep Russian titles in italics for authenticity
- Provide translation in parentheses on first appearance

### Slang
- Preserve register - if Russian is casual, English/Chinese/Japanese should be too
- Add translator notes for untranslatable terms

### Technical Accuracy
- Use industry-standard translations for tech terms
- Prefer English loan words in Chinese/Japanese where commonly used

---

## Translation Principles

1. **Consistency**: Use same translation for same term throughout
2. **Context**: Adjust register to match original
3. **Clarity**: Target audience is educated, tech-savvy
4. **Voice**: Preserve Durov's sharp, provocative tone
5. **Cultural bridge**: Make Russian concepts accessible
